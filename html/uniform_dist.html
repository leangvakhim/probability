Introduction

Mathematical Blueprint

Notation: X ~ U(a, b)
PDF Probability Density Function
f(x) =
1 b - a
Mean (μ):
a + b 2
Variance (σ²):
(b - a)² 12
CDF (Cumulative Distribution Function):
F(x) =
x - a b - a
(for a ≤ x ≤ b)

Applying it in Python

Instead of calculating by hand, data scientists use scipy.stats to automate this. Let's model our Subway example where wait times are Uniformly distributed between 0 and 10 minutes. 

from scipy.stats import uniform

# 1. Define U(0, 10)
# SciPy uses 'loc' for 'a' and 'scale' for 'b - a'
a = 0
scale = 10 - a
dist = uniform(loc=a, scale=scale)

# 2. PDF: Height of the distribution curve at x=5
pdf_val = dist.pdf(5)
print(pdf_val) # Output: 0.1

# 3. CDF: Probability of waiting 5 mins OR LESS
cdf_val = dist.cdf(5)
print(cdf_val) # Output: 0.5 (50% chance)

# 4. Mean: Expected average wait time
mean_val = dist.mean()
print(mean_val) # Output: 5.0 minutes

# 5. Variance: Spread of the wait times
var_val = dist.var()
print(var_val) # Output: 8.333...
Pro-Tip: The cdf() method is perfect for finding probability ranges. What's the chance of waiting between 3 and 5 minutes? Just subtract them:

dist.cdf(5) - dist.cdf(3) = 0.2 (20%)
